# Changelog

All notable changes to FreezR will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Planned for v0.1.0 (MVP)
- Basic process scanning from /proc
- SIGSTOP/SIGCONT freeze/unfreeze mechanism
- TOML configuration system
- Systemd service integration
- CLI tool for monitoring and control
- Configurable thresholds for CPU and memory
- Protected process list
- Logging with tracing

### Planned for v0.2.0 (Intelligence)
- Thermal monitoring
- System health metrics
- Profile system (gaming, development, power-saving)
- Desktop notifications
- ML prediction (experimental)
- Emergency mode for critical situations

### Planned for v1.0.0 (Production)
- Cgroup v2 integration
- Web dashboard
- Desktop GUI
- Plugin system
- Advanced analytics
- Multi-user support

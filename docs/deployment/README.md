# FreezR Deployment Guides

Documentation for deploying FreezR in production environments.

## Available Guides

### [Systemd Integration](systemd.md)
Complete guide for running FreezR as a systemd service:
- Installation steps
- Service file configuration
- Service management (start, stop, restart)
- Log viewing with journalctl
- Advanced configuration (multiple instances, resource limits)
- Monitoring service health
- Troubleshooting
- Complete production setup example

**Recommended for production deployments on Linux with systemd.**
